# Fraud Detection Model

## 📊 Project Overview

This project focuses on identifying fraudulent transactions using machine learning techniques.

## 🚀 Problem Statement

Financial fraud leads to significant losses. The goal is to build a model that can accurately detect fraudulent activities in transaction data.

## 🧠 Techniques Used

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Handling Imbalanced Data
* Feature Engineering
* Classification Models (Logistic Regression, Decision Tree, Random Forest)

## 📂 Dataset

* Transaction dataset containing normal and fraudulent records

## ⚙️ Tools & Technologies

* Matplotlib, Seaborn
